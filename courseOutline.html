Description

  • Ordered lists: Used to create a list of items in a specific order. Each item is numbered sequentially.
  • Unordered lists: Used to create a list of items where the order does not matter. Each item is marked with a bullet point.
  • Description lists: Used to create a list of terms and their descriptions.
